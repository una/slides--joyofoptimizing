      How WebP Works: Predictive coding - both lossy and lossless variants
      Predictive coding uses the values in neighboring blocks of pixels to predict the values in a block, and then encodes only the difference
      image time WebP Lossless: https://developers.google.com/speed/webp/docs/webp_lossless_alpha_study#results WebP Lossy: https://developers.google.com/speed/webp/docs/webp_study
